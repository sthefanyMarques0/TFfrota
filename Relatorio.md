# Descrição do projeto

Este projeto é um sistema de Gerenciamento de Frota feito em Java usando orientação a objetos e uma interface gráfica com Swing. Ele permite cadastrar, listar e remover veículos, mostrando tudo em uma tabela.

O sistema conta com as seguintes funcionalidades como:

calcular a média de consumo da frota;

mostrar quais veículos precisam de manutenção;

indicar o veículo mais rodado;

exibir o total de cada tipo de veículo;

buscar e remover veículos pela placa;

carregar alguns veículos de exemplo e adicionar novos.
